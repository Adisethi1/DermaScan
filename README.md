# DermaScan
AI-Powered Skin Cancer Detection &amp; Diagnostic Dashboard. A decoupled 2-tier MLOps platform leveraging transfer learning (MobileNetV3 Small) on the benchmark HAM10000 dataset to classify 7 skin lesion types. Combines a PyTorch core with inverse-frequency class weighting, a Flask REST API backend, and an interactive Streamlit doctor-facing Web UI.
